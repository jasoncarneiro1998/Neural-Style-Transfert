Neural Style Transfert

Nous allons vous présenter une technique utilisant le Deep Learning pour appliquer à une image d’origine le style d’une autre. Cette technique d’optimisation est connue sous le nom de "Neural Style Transfer".

Vous pouvez tester l'expérience en déposant dans un premier temps une photo d'origine puis le style que vous souhaitez apporter à la photo.

Notre modèle de Deep Learning vous présentera le résultat obtenu!
